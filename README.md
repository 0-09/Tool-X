# installation in termux


Open the termux app and type following commands.

apt update

pkg install python

pkg install git

git clone https://github.com/Rajkumrdusad/Tool-X.git


Now Tool-X is installed successfully. To run Tool-X Type

cd Tool-X

python Tool-X.py
